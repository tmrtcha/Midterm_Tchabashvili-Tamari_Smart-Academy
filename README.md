# Next.js E-commerce Project

This project is a simple e-commerce web application built with Next.js App Router.  
It includes product listing, product details, cart system, and user profile page using Fake Store API.

---

## Live Features

- Products page with API data fetching
- Product details dynamic route (`/product/details/[id]`)
- Shopping cart with quantity control (1–10)
- Cart data stored in LocalStorage
- Profile page with server-side data fetching
- Reusable Navbar and Footer components
- Responsive grid layout
- Navigation between pages using Next.js App Router

---

## Tech Stack

- Next.js 16
- React 19
- JavaScript (ES6+)
- CSS Modules
- Fake Store API

---

## API Endpoints Used

- Products: https://fakestoreapi.com/products  
- Product Details: https://fakestoreapi.com/products/:id  
- Cart: https://fakestoreapi.com/carts/1  
- User: https://fakestoreapi.com/users/3
